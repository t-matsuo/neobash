### string::start_with

check if the variable is start with specified string or not

#### Options

* **--string \<string\>**

  Target string. (required)

* **-s \<string\>**

  Alias for --string

* **--match \<string\>**

  String to match. (required)

* **-m \<string\>**

  Alias for --match

#### Exit codes

* **0**: Matched.
* **1**: Not matched.

#### Output on stdout

* None.

#### Output on stderr

* None.
